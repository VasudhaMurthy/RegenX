# ♻️ ReGenX - Revolutionizing waste, regenerating resources 🚮✨

> 🚀 A smart waste tracking and segregation system built to revolutionize how cities manage garbage — using AI, geofencing, and people power.

---

## 🌟 Why ReGenX?

Because the planet deserves better 🌍  
ReGenX empowers citizens, garbage collectors, and officials to **track, report, and recycle** — all through a smart Android app built for real impact.

## 🧠 Features

- 📍 **Dynamic Geofencing**: Know exactly when the garbage truck is near you!
- 🧠 **AI Waste Segregation**: Use AI to identify and sort waste types.
- 🔐 **Firebase Auth**: Secure login for every role.
- 🎙️ **Multilingual Support**: Regional audio & voice features.
- 📸 **Proof-based Grievance Reporting**: Upload images/videos of waste violations.
- 💬 **Chatbot for Waste Queries**: Ask the AI — “Is this recyclable?”
